Verify a draw

This page re-fetches the committed EOS block straight from the public EOS blockchain and recomputes the winner & prize using the exact same formula the draw used. Nobody can fake it: the block hash didn't exist when the block number was committed. This page's full source is public on GitHub, so you can read it or run the math yourself.

The formula 
key    = sort(entrants) joined by ","
N      = number of entrants
winner = sort(entrants)[ SHA256(blockhash + ":winner:" + key) mod N ]
prize  = weighted( SHA256(blockhash + ":prize:" + key) mod 100 )
weights: 5 Coins 60, 10 Coins 20, Battle Call 18, 25 Coins 2
